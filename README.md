# — PROJET HTML/CSS/JS + PHP 
        Pauline Fauvel


Ce projet est une application web qui permet à un utilisateur de :
 - Créer un compte
 - Se connecter
 - Accéder à ses informations
 - Un système de role (user + admin)
 - Un admin peut modifier les roles et supprimer des comptes
 - Une auto complétation des adresse grâce à l'API officielle du gouv. français.

Languages utilisés :

 - PHP (backend)
 - MySql via Xampp (base de données)
 - HTML + CSS (interface)
 - JavaScript (autocomplétation)
 - API : https://api-adresse.data.gouv.fr

Fonctionnalités : 
 - Inscription sécurisée avec mot de passe hashé
 - Connexion utilisateur
 - Vérification des emails en doublon
 - Autocomplétion de l’adresse lors de l’inscription
 - Système de rôles (admin / utilisateur)
 - Tableau de bord utilisateur
 - Menu responsive avec burger
 - Design responsive (mobile / tablette / PC)
 - Burger menu via CSS

Comment l'utiliser :
 - Cloner, ou télécharger les fichiers
 - Placer le projet dans le dossier `htdocs` pour Xampp ou `www` pour Wamp
 - Démarrer Apache et MySQL 
 - Importer la base de données
 - Configurer l'accès à la base dans fonctions.php
